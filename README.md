## 📖 项目简介

本项目致力于为重庆大学有意向转专业的同学提供全面的参考信息。我们整理了转专业流程、经验分享和注意事项，希望能帮助同学们在转专业过程中做出更明智的选择。

## 🔍 主要内容

- 转专业流程时间线与关键节点
- 各学院转专业要求与考核方式
- 学分认证与课程补修指南
- 过来人经验分享
- 常见问题解答
